# 项目介绍

这是一个基于 VitePress 的文档模板，旨在帮助你快速启动一个新的文档项目，同时保留现有的样式和功能配置。

## 包含的功能

- **基础配置**: 预设了中文支持、侧边栏结构和导航栏。
- **插件集成**:
  - `GitChangelog`: 自动显示文档的修改历史。
  - `PageProperties`: 显示阅读时间、字数统计等。
  - `EnhancedReadabilities`: 增强阅读体验。
- **美化组件**: 内置了 `ArticleMetadata` 等自定义组件。

## 为什么使用这个模板？

如果你觉得当前的 `docs` 文件夹太臃肿，或者想开启一个全新的主题，使用这个模板可以确保你不需要重新配置繁琐的 `config.mts` 和主题样式。
